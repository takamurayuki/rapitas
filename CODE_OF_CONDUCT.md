# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## 報告先 / Reporting

行動規範への違反、または不適切な行為を目撃した場合は、以下に報告してください。

If you experience or witness unacceptable behavior, please report it to:

- GitHub Issue (公開): プロジェクトの Issue として起票
- メール (非公開): リポジトリの `SECURITY.md` に記載された連絡先

報告は機密扱いとし、報告者の身元は公開しません。
All reports are confidential and reporter identity is protected.

## 適用範囲 / Scope

この行動規範は、本プロジェクトのリポジトリ、Issue、Pull Request、ディスカッション、外部チャンネルでの公式コミュニケーションに適用されます。

This Code of Conduct applies within all project spaces (repository, issues, PRs, discussions) and applies when an individual is officially representing the project in public spaces.
